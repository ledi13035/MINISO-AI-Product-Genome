# AI 架构详细设计

## 总体架构

系统采用**多智能体协同（Multi-Agent Collaboration）**架构，四个核心 Agent 围绕"产品基因"主题进行分工协作。

```
┌─────────────────────────────────────────────┐
│             用户需求入口                      │
│  (目标用户 / 品类 / 价格 / 关键词)            │
└────────────────────┬────────────────────────┘
                     ↓
┌─────────────────────────────────────────────┐
│         Orchestrator (调度器)                │
│   - 解析需求                                  │
│   - 拆解任务到 4 个 Agent                     │
│   - 汇总结果生成决策报告                      │
└────────────────────┬────────────────────────┘
                     ↓
        ┌────────────┼────────────┬────────────┐
        ↓            ↓            ↓            ↓
  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
  │ Trend    │ │ Consumer │ │ Design   │ │Prediction│
  │ Agent    │ │ Agent    │ │ Agent    │ │ Agent    │
  └────┬─────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘
       ↓            ↓            ↓            ↓
  ┌─────────────────────────────────────────────────┐
  │         Cultural Genome Library (文化基因库)      │
  │  - IP 潮流文化                                     │
  │  - 东方元素 (故宫/敦煌/宋画/节气)                  │
  │  - 情绪消费 (治愈/解压/亚比)                      │
  │  - 年轻网络文化 (B站/小红书/二次元)                │
  └─────────────────────────────────────────────────┘
                     ↓
              ┌──────────────┐
              │ 决策报告输出  │
              └──────────────┘
```

## 四个 Agent 详细设计

### 1. Trend Agent（趋势智能体）

**职责：** 实时感知市场趋势变化

**输入：**
- 类目（生活家居 / 美妆 / 饰品 等）
- 平台（淘宝 / 小红书 / 抖音）
- 时间窗口

**输出：**
- 趋势关键词 Top 20
- 热度评分（0-100）
- 增速指标（环比 / 同比）

**实现思路：**
- 数据源：小红书笔记 API、抖音热榜、淘宝搜索指数
- 关键词提取：TF-IDF + 情感分析
- 趋势评分：滑动窗口 + 异常检测

### 2. Consumer Agent（用户智能体）

**职责：** 解析目标用户画像与情绪需求

**输入：**
- 年龄、性别、城市层级
- 场景（送礼 / 自用 / 收藏）
- 关键词

**输出：**
- 用户画像（人群标签、消费力、兴趣偏好）
- 情绪价值点 Top 5
- 痛点与未被满足的需求

**实现思路：**
- 标签体系：基于名创优品历史用户数据训练
- 情绪分析：BERT 微调分类
- 需求挖掘：评论语料主题模型

### 3. Design Agent（设计智能体）

**职责：** 调用文化基因库，生成产品设计语言

**输入：**
- 关键词（治愈、东方、年轻化）
- 风格标签
- 预算

**输出：**
- 设计元素（纹样、配色、形态）
- 推荐材料
- 草图 prompt

**实现思路：**
- 文化基因库：4 大类目 × 100+ 子标签的结构化知识图谱
- 重组算法：基于向量相似度的"基因配对"
- 视觉生成：Stable Diffusion + ControlNet

### 4. Prediction Agent（预测智能体）

**职责：** 综合信号预测产品爆款概率

**输入：**
- Trend / Consumer / Design Agent 输出
- 历史 SKU 销售数据
- 价格带竞争密度

**输出：**
- 爆款概率（0-100%）
- 风险提示
- 预期销售区间

**实现思路：**
- 特征工程：40+ 维度（趋势热度、用户匹配、文化基因重合度、价格敏感度）
- 模型：XGBoost + LightGBM 集成
- 训练数据：名创优品近 3 年 SKU 销售数据

## Cultural Genome Library 文化基因库

### 数据结构

```json
{
  "id": "genome_001",
  "category": "东方元素",
  "sub_category": "故宫纹样",
  "name": "海水江崖纹",
  "description": "传统帝王服饰下摆纹样，象征山河壮丽",
  "applicable_products": ["文创礼品", "服饰", "家居装饰"],
  "color_palette": ["#C8102E", "#FFD700", "#003366"],
  "age_group": "18-45",
  "trend_score": 78,
  "examples": [
    "故宫文创书签",
    "新中式连衣裙"
  ]
}
```

### 四大类目

| 类目 | 标签数 | 代表元素 |
| --- | --- | --- |
| IP 潮流文化 | 50+ | Disney、Sanrio、Loopy、Chiikawa |
| 东方元素 | 80+ | 故宫纹样、敦煌壁画、宋画、节气 |
| 情绪消费 | 40+ | 治愈、解压、亚比风、Citywalk |
| 年轻网络文化 | 60+ | B站梗、抖音爆款、二次元 |

## 调度协议

Orchestrator 通过**消息队列**与各 Agent 通信：

```
[Orchestrator] → Trend Agent.run(品类, 平台) → result_trend
[Orchestrator] → Consumer Agent.run(用户, 关键词) → result_consumer
[Orchestrator] → Design Agent.run(关键词, 风格, 趋势) → result_design
[Orchestrator] → Prediction Agent.run(trend, consumer, design) → result_prediction
[Orchestrator] → assemble_report(...) → final_report
```
